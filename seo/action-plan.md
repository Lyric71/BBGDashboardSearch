## PHASE 6 — Technical SEO (Week 1–3)
*Ensure Google can fully crawl, index, and understand every page*

### 6.1 Schema markup audit
**Current:** CBEC page has FAQPage + WebPage + Organization (good)
**Missing on other pages:**
- Homepage: Organization + LocalBusiness + Sitelinks schema
- Service pages: Service + FAQPage + BreadcrumbList
- Blog articles: Article + Author + BreadcrumbList
- Case studies: Article + Review (if testimonial present)

### 6.2 Internal linking — connect new pages into the site architecture

**Minimum internal link plan (for new pages):**
- Homepage → `/china-ecommerce-consulting/` (in nav + body)
- Homepage → `/china-ecommerce-agency/` (in nav + body)
- `/china-ecommerce-consulting/` → `/china-ecommerce-agency/` (sidebar or body)
- Both pages → `/china-cross-border-ecommerce-entry-consulting/` (footer link or related services)
- Every blog article → the relevant pillar page (minimum 2 internal links per article)
- `/cbec-agency/` → `/china-cross-border-ecommerce-entry-consulting/` (main CTA)

### 6.3 URL structure — enforce clean slugs
Current issue: BBG has long, descriptive URLs (e.g., `/tmall-ecommerce-agency-in-china-sell-your-products-in-china/`)
All new pages should follow the clean slug pattern already proven on the CBEC consulting page.
- ✓ `/china-cross-border-ecommerce-entry-consulting/` — good
- ✓ `/china-ecommerce-consulting/` — new, clean
- ✗ `/tmall-ecommerce-agency-in-china-sell-your-products-in-china/` — create a redirect to `/tmall-consulting/` or `/tmall-agency/`

### 6.4 Page speed
Ensure new pages load in under 2.5s on mobile (Core Web Vitals: LCP ≤ 2.5s, CLS < 0.1, INP ≤ 200ms). Logo strips and badge images are the most common speed culprits — use WebP, lazy-load, specify dimensions.

### 6.5 Hreflang (EN/FR)
BBG operates in English and French markets. For every new page:
- Create an FR equivalent (or at minimum, an FR meta alternate tag)
- Implement `hreflang="en"` and `hreflang="fr"` in the head of each page pair
- French keyword opportunity: "agence ecommerce chine", "consulting ecommerce chine", "CBEC chine" — all low competition
